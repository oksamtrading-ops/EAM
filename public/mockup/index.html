Changes Summary

Sidebar: Removed linear-gradient(180deg, #0a0e1a, #111827) inline style. Now uses .glass-sidebar utility — translucent frosted glass with backdrop-filter: blur(40px). Background content subtly visible through sidebar.

Colors: All text-white/50, bg-white/[0.06], bg-[#0B5CD6] replaced with semantic tokens: text-muted-foreground, hover:bg-muted/50, bg-primary. Works in both light and dark mode automatically.

Borders: All 3 inline style={{ borderTop/Bottom: "1px solid rgba(255,255,255,0.08)" }} replaced with border-t border-border / border-b border-border CSS classes.

Mobile header: bg-white/80 backdrop-blur-md replaced with glass-toolbar utility.

Tooltips: bg-[#1a1f2e] text-white replaced with bg-foreground text-background (auto dark-mode).

Transition: Sidebar width uses ease-[var(--ease-out-expo)] for smoother expand/collapse.

Nav hover: No more 3D rotateY. Simple translateX(3px) via data-active attribute selector in globals.css.

Files modified: DashboardShell.tsx, WorkspaceSwitcher.tsx, PageSearchTrigger.tsx
